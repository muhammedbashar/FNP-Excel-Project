# Sales Analysis Dashboard
### Project Overview

This project analyses sales performance for an occasion-based gifting business using an interactive dashboard. The goal is to uncover key revenue drivers, customer purchasing behaviour, seasonal trends, and high-performing products to support data-driven business decisions.

* Product demand is diversified across multiple top-performing items

### Dashboard Highlights

* Revenue analysis by occasion, category, month, hour, and city

* Top products and customer spending metrics

* Interactive filters for order date, delivery date, and occasion

### Tools Used

* Microsoft Excel

* Pivot Tables & Charts

* Slicers for interactivity

* Data cleaning and transformation

### Business Value

This dashboard helps stakeholders understand sales patterns, optimise marketing timing, plan inventory efficiently, and identify high-growth regions and products.
